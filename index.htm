Dynamic Pie Chart Maker

The goal of this project is to create and intuitive dynamic pie charts any user can make with absolutely no coding required, simply by inputting data. Input raw data and watch the pie chart update right in front of you.

Things left to implement: making pie chart size dynamic in Canvas, being able to download the pie chart.
